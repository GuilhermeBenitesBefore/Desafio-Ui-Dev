Olá.

Fizemos esse setup inicial para você focar no que interessa, que são os seguintes pontos:

  • Organização do código;
  • Utilização correta dos recursos que o HTML e SASS oferecem;

mas se você quiser mudar toda a estrutura de arquivos, nomes e até a biblioteca de build, fique à vontade.

Orientações

  • Quando terminar, tire um print da sua tela e coloque na pasta, "seu-print-aqui";
  • Verifique se está tudo ok e faça seu commit com todos os arquivos que você criou e alterou;
  • Por último, faça um pull request para o repositório base.

Boa sorte!
